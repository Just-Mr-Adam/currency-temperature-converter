# Конвертер единиц измерения
**Простое desktop-приложение на Windows Forms (.NET) для конвертации валют и температур.**

**Возможности**
- Переключение между двумя режимами:
  - Конвертер валют (USD, EUR, RUB, GBP и другие популярные валюты)
  - Конвертер температур (°C ↔ °F ↔ K)
- Удобный интерфейс с выпадающими списками единиц измерения
- Мгновенный расчёт при изменении значений
- Исполование API для получения актуальной информации

**Требования**
- Windows 10 или новее
- .NET 10.0 или выше

**Как запустить**
- Скачайте или склонируйте репозиторий
- Откройте exe файл по пути currency-temperature-converter\CurrencyApp\bin\Debug\net10.0-windows/CurrencyApp.exe
